# Clean code by Hymaïa

## Objectif de l'exercice

Dans le fichier `main.py` vous trouverez 2 fonctions avec une logique boolean dedans. Ces fonctions sont complexes.

Trouvez un moyen pour les rendre les plus lisibles possible, sans en changer le comportement.

## Comment lancer le projet

### Prérequis

installer :
* uv >= 0.5.7
* python 3.12

### Commande d'exécution

```bash
uv run main
```
