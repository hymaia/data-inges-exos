# Clean Code by Hymaïa

## Objectif de l'exercice

Voici un projet complet Python / Spark. Celui-ci contient plusieurs job Spark différents :

```text
src.fr.hymaia.exo1.main:main
src.fr.hymaia.exo2.spark_clean_job:main
src.fr.hymaia.exo2.spark_aggregate_job:main
src.fr.hymaia.exo4.python_udf:main
src.fr.hymaia.exo4.scala_udf:main
src.fr.hymaia.exo4.no_udf:main
```

Vous trouverez aussi des tests unitaires.

Votre travail est d'améliorer tout ce qui vous semble pas bon pour en faire le projet le plus impeccable selon le context que vous aurez vous-même défini.
